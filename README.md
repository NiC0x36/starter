# Starter

## Requirements

- node
- yarn or npm

## Installing / Getting started

Install NPM-Modules via

```shell
yarn install
# or
npm install
```

This install the required node_modules.
Then run

```shell
# Run in development mode witch hotreload
yarn dev
# or
npm run dev

# Build production files
yarn build
# or
npm run build

# Clean development base
yarn clean
# or
npm run clean
```
